## Prompt 5 (Claude3): AWS Infrastructure with Terraform

> Implement AWS infrastructure with Terraform for minimal cost deployment:
> - S3 + CloudFront for frontend (static export)
> - EC2 t3.micro with docker-compose (backend + postgres + redis)
> - No ALB, no RDS, no ElastiCache (cost savings)
> - Default VPC (no NAT Gateway)
> - Route53 with feature.rochatecnologia.com.br (frontend) and api.feature.rochatecnologia.com.br (backend)
> - ACM certificate for HTTPS
> - CloudFront function for SPA routing
> - Auto-generated SSH key pair
> - Estimated cost: ~$0/month (free tier) or ~$11/month after

## Prompt 6 (Claude3): GitHub Actions CD Workflows

> Create CD (Continuous Deployment) GitHub Actions workflows for backend, frontend, and Android app:
> - Backend CD: test -> deploy to EC2 via rsync + docker compose rebuild, with health check
> - Frontend CD: lint + build static export -> sync to S3 -> invalidate CloudFront cache
> - Android CD: test -> build release APK -> create GitHub Release with APK attached
> - All triggered on push to main with path filters
